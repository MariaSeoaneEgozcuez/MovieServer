## Para inciar el servidor

Desde la carpeta principal del proyecto ejecutar el siguiente comando:

```
npm start
```

Despues de que se inicia el proceso en el puerto 3000 nos vamos a la carpeta de /moviefrontend:

```
cd ./moviefrontend
```

Dentro de esta carpeta sin haber parado el proceso anterior ejecutamos el siguiente comando:

```
npm run start
```
